Please note: SertexOS 2: Server Edition requires a installer to connect to github! Enter this as your http_whitelist:
```
S:http_whitelist=*.pastebin.com;*.github.com;*.githubusercontent.com
```
or you can use
```
S:http_whitelist=*
```
